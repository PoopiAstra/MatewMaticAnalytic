# GED0055 Calculus Reviewer

A responsive study website for GED0055 Modules 1–7.

## Included

- Seven expanded, plain-language module reviewers
- Twenty questions per module
- A 140-question comprehensive test
- Canvas, Human, and Mixed quiz formats
- Balanced answer positions across A, B, C, and D
- Mathematical notation rendered with MathJax
- Instant feedback and explanations in Mixed mode
- Local best-score tracking
- Persistent Spotify study player
- Desktop and mobile navigation
- Loading and question-transition animations

The website is a static browser app. Scores, preferences, and the saved Spotify link stay in the current browser through local storage.
